## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочкой (*), например:
* Строка 1
* Строка 2
* Строка 3

Чтобы добавить нумерованные списки, необходимо пункты пронумеровать. Например: вот так

1. Первая строка
2. Вторая строка
3. Третья строка
